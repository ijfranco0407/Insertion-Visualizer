Algorithm Description

Insertion Sort is a simple sorting algorithm that builds the final sorted array one item at a time. It is much less efficient on large lists than more advanced algorithms such as quicksort, heapsort, or merge sort.

How it works:

  • Start with the second element (index 1)
  • Compare it with the elements before it
  • Shift greater elements one position up
  • Insert the current element in its correct position
  • Repeat for all elements

Time Complexity:

Case Complexity
Best O(n)
Average O(n²)
Worst O(n²)

Implementation 

function insertionSort(arr) {
    for (let i = 1; i < arr.length; i++) {
        let key = arr[i];
        let j = i - 1;
        
        // Move elements that are greater than key
        // to one position ahead of their current position
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
    return arr;
}

The algorithm works by:

  1. Taking each element from the array (starting from index 1)
  2. Comparing it with the elements before it
  3. Moving larger elements one position up
  4. Placing the current element in its correct position
